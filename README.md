## 📖 Описание проекта

**GitHub User Search & Random Quote Generator** — это десктопное GUI-приложение на Python, которое объединяет два функциональных модуля:

1. **Поиск пользователей GitHub** через официальное GitHub API
2. **Генератор случайных цитат** с возможностью фильтрации и сохранения истории

Приложение разработано в рамках итоговой аттестации по курсу программирования и демонстрирует навыки работы с:
- GUI-фреймворком Tkinter
- HTTP-запросами (библиотека `requests`)
- REST API (GitHub API)
- JSON-файлами для хранения данных
- Многопоточностью для асинхронных операций
- Валидацией пользовательского ввода
- Модульным тестированием

## 🚀 Основные возможности

### Модуль 1: Поиск пользователей GitHub 🔍

| Функция | Описание |
|---------|----------|
| **Поиск пользователей** | Введите имя пользователя и получите список релевантных аккаунтов |
| **Валидация ввода** | Проверка формата имени (буквы, цифры, дефисы, длина 1-39 символов) |
| **Асинхронные запросы** | Поиск выполняется в отдельном потоке, GUI не зависает |
| **Детальная информация** | При клике на пользователя отображается: имя, компания, локация, био, статистика |
| **Просмотр репозиториев** | Отображение публичных репозиториев с количеством звёзд и форков |
| **Открытие профиля** | Быстрый переход на GitHub профиль пользователя в браузере |
| **Обработка ошибок** | Полная обработка HTTP ошибок, таймаутов и проблем с соединением |

### Модуль 2: Генератор цитат 🎲

| Функция | Описание |
|---------|----------|
| **Случайная цитата** | Генерация случайной цитаты из коллекции одним кликом |
| **Фильтрация** | Фильтрация по автору и/или теме |
| **Добавление цитат** | Возможность добавлять собственные цитаты с валидацией |
| **История просмотров** | Автоматическое сохранение всех сгенерированных цитат с временными метками |
| **Фильтрация истории** | Поиск в истории по автору или теме |
| **Очистка истории** | Полная очистка истории с подтверждением |
| **JSON-хранилище** | Все данные сохраняются в файлы quotes.json и history.json |

## 💻 Технические требования

### Необходимое ПО
Python 3.8 или выше
PIP (менеджер пакетов Python)
Git (опционально, для клонирования репозитория)
Интернет-соединение (для работы с GitHub API)

### Поддерживаемые OC
Windows 10/11
macOS 10.15+
Linux (Ubuntu 20.04+, Debian 11+, Fedora 35+)

### Установка и запуск
# Клонируйте репозиторий
git clone https://github.com/ivanov/github-user-search-app.git

# Перейдите в директорию проекта
cd github-user-search-app

# Установите зависимости
pip install -r requirements.txt

# Запустите приложение
python main.py
