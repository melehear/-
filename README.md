## Описание

Генератор случайных цитат — это десктопное GUI-приложение, созданное на Python с использованием Tkinter, которое предоставляет вдохновляющие и глубокомысленные цитаты от различных авторов на разные темы. Приложение позволяет пользователям генерировать случайные цитаты, фильтровать по автору или теме, вести историю просмотренных цитат и добавлять новые цитаты в коллекцию.

### Возможности

- **Генерация случайных цитат**: Получайте случайные цитаты одним кликом
- **Система фильтрации**: Фильтруйте цитаты по автору и/или теме
- **Отслеживание истории**: Автоматически сохраняет все сгенерированные цитаты с отметками времени
- **Управление цитатами**: Добавляйте новые цитаты в коллекцию с проверкой ввода
- **Постоянное хранение**: Цитаты и история сохраняются в формате JSON
- **Чистый интерфейс**: Удобный интерфейс с интуитивно понятными элементами управления
- **Проверка ввода**: Предотвращает пустые поля при добавлении новых цитат

### Технические детали

- **Язык**: Python 3.8+
- **GUI-фреймворк**: Tkinter (встроенный)
- **Хранение данных**: JSON-файлы
- **Зависимости**: Отсутствуют (используется только стандартная библиотека Python)
